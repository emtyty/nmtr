# Keyboard Shortcuts

## Global Shortcuts

These shortcuts work from anywhere in the application window.

| Shortcut | Action |
|---|---|
| `Ctrl+Enter` | **Start / Stop** the active trace session |
| `Ctrl+R` | **Reset** — clear all stats for the active session without stopping the probe loop |
| `Ctrl+E` | **Export as Text** — generate a WinMTR-compatible text report and copy it to the clipboard |
| `Ctrl+,` | **Open Settings** |

## Dialog Shortcuts

| Shortcut | Action |
|---|---|
| `Escape` | Close the currently open dialog (Settings, WHOIS, Latency Detail) |

## Planned / Not Yet Assigned

The following actions are accessible via UI controls but do not yet have keyboard shortcuts:

- New trace session
- Switch active session
- Open recording file
- Start / stop recording
- Export as CSV / HTML

> If you would like to suggest additional shortcuts, open an issue on the [GitHub repository](https://github.com/your-org/nmtr/issues).
